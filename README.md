# 🚖 Taxi Trips Analysis: Сравнительный анализ тарифов Economy и Comfort

Проект посвящен разведочному анализу данных (EDA) сервиса такси и проверке статистических гипотез. Основная цель — определить, влияют ли тарифные предпочтения пользователей на дистанцию поездок, и доказать наличие или отсутствие значимых различий между выборками.

## 🎯 Бизнес-задача
Проверить, существует ли у пользователей явное предпочтение более комфортных условий (класс `Comfort`) при поездках на дальние расстояния по сравнению с классом `Economy`.

## 🛠️ Стек инструментов
* **Язык:** Python
* **Библиотеки для анализа данных:** Pandas, NumPy
* **Визуализация:** Matplotlib, Seaborn
* **Статистический анализ:** SciPy (`scipy.stats`)

## 📊 Ход исследования и ключевые этапы

### 1. Первичный анализ данных (EDA)
* Загружен датасет объемом 2000 записей со следующими ключевыми метриками: `distance_km` (дистанция), `offer_class_group` (тариф), `driver_response` (статус заказа) и др.
* Изучена описательная статистика: средняя дистанция поездок по всему датасету составляет **26.28 км**, а медианное значение — **22.83 км**.
* Были сформированы две сбалансированные выборки одинаковой длины для классов `Economy` и `Comfort`.

### 2. Визуализация данных
Построены графики плотности распределения дистанций для обоих тарифов (sns.histplot с оценкой плотности ядра kde=True). Визуальный анализ выявил:

Оба распределения имеют явный логнормальный характер и характеризуются выраженной правосторонней асимметрией (длинным «хвостом» в сторону больших дистанций). Это логично для специфики работы такси: подавляющее большинство поездок совершается на короткие и средние дистанции, а сверхдлинные заказы единичны.

В связи с сильным отклонением формы распределений от нормального (параметрического) вида, классический t-критерий Стьюдента для проверки гипотез неприменим.

### 3. Проверка статистических гипотез
Для анализа был выбран непараметрический **U-критерий Манна — Уитни** (односторонний тест) с уровнем значимости $\alpha = 0.05$.

* **$H_0$:** Для дальних поездок пользователи не имеют явных предпочтений между тарифами.
* **$H_1$:** На дальние поездки пользователи предпочитают тариф `Comfort` тарифу `Economy`.

**Результаты теста:**
* `p-value` = **0.00064246**
* Так как `p-value` значительно меньше уровня значимости $\alpha$ ($0.00064 < 0.05$), **нулевая гипотеза была уверенно отвергнута**.

## 📈 Итоговый вывод
Статистический тест строго доказал, что распределение тарифа `Comfort` значимо смещено в сторону более длинных дистанций по сравнению с `Economy`. В масштабах всего сервиса пользователи действительно **чаще выбирают повышенный класс комфорта, если им предстоит дальняя поездка**.
